# 📊 **Store Sales Dashboard – Project Documentation**

## 1. 📌 Project Overview

This project focuses on analyzing e-commerce sales data using **Microsoft Excel** to derive business insights and support decision-making.

The dashboard provides a clear view of:

* Sales performance over time
* Channel-wise contribution
* Customer demographics
* Regional sales distribution
* Order status analysis

---

## 2. 🎯 Objective

The main goal of this project is to:

* Identify **top-performing sales channels**
* Analyze **customer behavior (age & gender)**
* Track **monthly sales trends**
* Detect **business issues (returns, cancellations)**
* Provide **data-driven recommendations**

---

## 3. 📂 Dataset Description

The dataset contains transactional-level data with fields such as:

* Order ID
* Order Date / Month
* Sales Amount
* Channel (Amazon, Flipkart, etc.)
* Category (Saree, Set, etc.)
* State
* Gender
* Age Group
* Order Status (Delivered, Cancelled, Returned)

---

## 5. 📊 Dashboard Insights

### 🔹 Sales Trend

* Peak sales observed in **Feb–Mar**
* Gradual decline after mid-year

### 🔹 Channel Performance

* **Amazon (~35%)** is the top contributor
* Myntra & Flipkart also significant
* Ajio & Others have low contribution

### 🔹 Order Status

* ✅ Delivered: ~92%
* ⚠️ Returned: ~3%
* ❌ Cancelled: ~3%
* 🔁 Refunded: ~2%

👉 Insight: Return/cancellation rate is low but still impacts revenue.

### 🔹 Top States

* Maharashtra (~3M) is highest
* Karnataka & Uttar Pradesh also strong

### 🔹 Customer Demographics

* Women: ~64% (major buyers)
* Men: ~36%

### 🔹 Age Group Analysis

* Adults contribute highest orders
* Women dominate across all age groups

---

## 6. ❗ Problems Identified

1. **Sales Decline After Peak Months**

   * Possible seasonal dependency

2. **High Dependency on Few Channels**

   * Amazon dominates → risk if performance drops

3. **Returns & Cancellations**

   * Even small % leads to revenue loss

4. **Limited Male Customer Contribution**

   * Untapped market segment

---

## 7. ✅ Solutions & Recommendations

### 📌 Business Solutions

* Increase marketing during **low-performing months**
* Diversify channel strategy (boost Flipkart, Myntra)
* Improve product quality & descriptions → reduce returns
* Target male customers with tailored campaigns
* Expand high-performing states strategy to other regions
